1.                 General Notes

This document contains the latest information concerning the present version of ManuWare.

ManuWare – Installation
ManuWare doesn't need to be installed necessarily via the installer routine. It is sufficient to copy the content of the directory 'ManuWare' to any directory on your local disc or on a network drive. ManuWare is even running on a memory stick.

User Manual
ManuWare 1.0 comes with a online help file, that is equivalent to the short instructions manual. It can be found in the 'documents'-directory in various languages.

This document lists amendments to the short instructions manual / online help file. Known problems are also listed here.

2.                 System Requirements

Minimal system requirements for ManuWare operation:

·         PC with minimum 500 MHz clock rate

·         Minimum of 128 MByte RAM

·         One RS-232C-COM-ports or USB-RS 232 converter/adapter

·         Windows 2000, Windows XP, Windows Vista operating instruction

4.                 Supported Devices

Firmware and flashloader-programs of the supported devices are found on the CD in the 'Supported Devices'-directory. The flashloader programs allow firmware upgrade to the latest firmware version found in the table above.

ICAM 5073:
Important: A firmware upgrade of ICAM 5073A... with the present flashloader program is only possible from firmware version 1.09. ICAM's with older firmware versions can only be updated at the factory.

Transmitter amplifiers:
Besides the transmitters 9238A... and 9337A... the special amplifier 5031AQ01 is running with exactly the same firmware. Firmware update can be done exactly the same way as with the transmitters.
